## SI522A

* QFN-32-EP(5x5)
* 13.56MHz 频段
* 支持 ISO/IEC 14443 A/MIFARE
* 超低功耗自动载波侦测功能（ACD）
    * 自动检测 13.56MHz 外部射频场和射频卡
    * ACD 过程不需要 MCU 干预
    * OSC 起振失败监测功能
典型操作距离为 50mm
* 支持的主机接口：
    * SPI 接口，速率高达 10Mbits/s
    * I2C 接口，快速模式速率达400kBd，高速模式速率达 3400kBd
    * 串行 UART，速率达 1228.8kBd
